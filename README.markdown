Installation
================================

    git clone git://github.com/chrisconley/houdini-sinatra-example.git
    cd houdini-sinatra-example
    gem install bundler -v 1.0.0.rc.5
    bundle install

    Change "HOUDINI_API_KEY" constant in app.rb to your api key

To run the app
--------------------------------

    cd houdini-sinatra-example
    rackup

To run the tests
--------------------------------

    cd houdini-sinatra-example
    ruby test.rb
